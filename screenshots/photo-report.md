# Final Project: Telecommunication Networks  
**Software-Defined Networking (SDN) using Ryu Controller + Mininet Emulator**

This repository presents the final project for the Telecommunication Networks course.  
We implemented and evaluated an SDN-based network using:  
- **Mininet** → Network emulation  
- **Ryu** → SDN controller (OpenFlow)  

Key features tested: topology creation, flow rules, traffic monitoring, basic routing/firewall/QoS experiments (depending on your implementation).
